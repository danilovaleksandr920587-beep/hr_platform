# lab-career.ru (CareerLab)

Next.js 16 платформа для джунов и стажеров: вакансии, база знаний,
AI-анализатор резюме, личный кабинет. Прод: https://lab-career.ru

## Документация (docs/) - читать ВМЕСТО сканирования кода

- `docs/OVERVIEW.md` - стек, структура репо, точка входа
- `docs/PAGES.md` - все страницы: URL, файл, назначение, middleware
- `docs/API.md` - все API-роуты: методы, auth, таблицы
- `docs/DATABASE.md` - схема БД, env-переменные
- `docs/FEATURES.md` - фичи по модулям, юзкейсы
- `docs/DEPLOY.md` - сервер, деплой, диагностика

## Правила

- Добавил/удалил страницу, API-роут или таблицу - обнови docs/ в том же коммите
- Проверка соответствия docs/ и app/: `npm run docs:check`
- Не использовать длинное тире в текстах, только дефис
