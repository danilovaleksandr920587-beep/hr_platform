# Фичи по модулям

## Вакансии
- Каталог с фильтрами: сфера, опыт, формат работы, тип занятости, зарплата
- Полнотекстовый поиск на русском (tsvector в Postgres)
- Карточка вакансии: `components/VacancyCard.tsx`, страница - SSG
- Сохранение в кабинет (сердечко), избранные вакансии на главной
- Наполнение: python-парсер `junior_jobs_parser/` + ручная модерация

## База знаний (SEO-ядро сайта)
- Статьи по 6 кластерам: резюме, собеседование, тестовые, зарплата, отклики, карьера
- Хаб-страницы категорий с приоритетами (`lib/knowledge-clusters.ts`)
- Фильтры по категории и уровню (Новичок/Продвинутый)
- Пайплайн публикации: `article_pipeline.py` (соседний проект /Users/user/claude,
  правила SEO - в его CLAUDE.md)
- `llms.txt` со списком всех статей для LLM-краулеров

## Инструменты
- **Анализатор резюме**: загрузка PDF/DOCX или вставка текста -> парсинг ->
  YandexGPT -> балл + чек-лист рекомендаций. Для залогиненных история
  сохраняется в кабинет
- **Калькулятор зарплат**: ориентиры по направлению/грейду/городу

## Авторизация
- Email + пароль (hash в `careerlab_accounts`)
- Яндекс OAuth (`/api/auth/yandex` -> `/auth/callback`)
- Восстановление пароля: токен + письмо по SMTP
- Сессия: JWT (jose) в HTTP-only cookie, 30 дней

## SEO-инфраструктура
- SSG для вакансий и статей, sitemap-индекс с секциями
- 308-редиректы: www -> без www, legacy `.html`-URL -> новые маршруты
- Канонический домен: lab-career.ru
