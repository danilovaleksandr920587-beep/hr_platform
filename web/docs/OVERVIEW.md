# lab-career.ru (CareerLab) - обзор проекта

Платформа для студентов и джунов: поиск стажировок и первой работы в IT.
Модули: вакансии, база знаний (SEO-статьи), AI-анализатор резюме, калькулятор
зарплат, исследования рынка, личный кабинет.

## Стек

- Next.js 16 (App Router), React 19, TypeScript
- Supabase (self-hosted на Beget) - контент: вакансии, статьи
- Postgres напрямую (пакет `postgres`) - пользователи, профили, сохранёнки
- Auth своя: JWT (jose) в HTTP-only cookie на 30 дней + Яндекс OAuth
- YandexGPT - анализ резюме
- nodemailer/SMTP - письма восстановления пароля
- Хостинг: VPS 155.212.216.117, nginx + pm2 (см. DEPLOY.md)

## Структура репозитория

```
hr_platform/
  web/                  <- ПРИЛОЖЕНИЕ (Next.js), всё живое здесь
    app/                страницы и API-роуты (см. PAGES.md, API.md)
    components/         компоненты, сгруппированы по модулям
    lib/                auth, data-слой, supabase-клиенты, db
    docs/               эта документация
    scripts/            служебные скрипты
  supabase/             миграции и seed (только vacancies + articles)
  junior_jobs_parser/   python-парсер вакансий (отдельный инструмент)
  *.html, *.css         СТАРАЯ статичная версия сайта, не используется
```
